Get Available Time Zone Names

       Home
       Import function and init.
       
        import {getTimeZoneNames} from 'noda-tz';
        const tzname = getTimeZoneNames()
        
       Get by:
        
        1. all == tzname.all == all available time zone names of IANA updated version.
        2. africa == tzname.africa == continent, africa.(some region tz will add later).
        3. asia == tzname.asia == continent, asia(some region tz will add later).
        4. america == tzname.america == continent, america. include (us, brazil, canada, chile,mexico ).
        5. europe == tzname.europe ==  continent, europe .
        6. antarctica == tzname.antarctica ==  continent, antarctica.
        7. australia == tzname.australia ==  continent, australia .
     
        8. usa == tzname.usa ==  usa
        9. brazil == tzname.brazil ==  brazil
       10. canada == tzname.canada ==  canada
       11. chile == tzname.chile ==  chile
       12. mexico == tzname.mexico ==  mexico
      
       13. atlantic == tzname.atlantic ==  atlantic
       14. arctic == tzname.arctic == arctic
       15. indian == tzname.indian ==  indian
       16. pacific == tzname.pacific ==  pacific
